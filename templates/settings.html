Microsoft Integration Setup

To sync Outlook emails and Teams messages, you need to register an app in Azure.

  1. Go to portal.azure.com → Azure Active Directory → App registrations → New registration
  2. Name: Work Tracker — Account type: Single tenant
  3. Under API permissions, add these Delegated permissions:
    • User.Read
    • Mail.Read
    • Chat.Read
    • offline_access
  4. Copy the Application (client) ID and Directory (tenant) ID
  5. Add them to your .env file: 
    AZURE_CLIENT_ID=paste-client-id-here
AZURE_TENANT_ID=paste-tenant-id-here
  6. The first time you click Sync, a login URL will appear in your terminal — open it and sign in with your work account.
